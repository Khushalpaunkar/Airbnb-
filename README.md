# 🏡 Airbnb 

A full-stack Airbnb Clone built using the MERN stack and MVC architecture. Users can browse listings, create accounts, add properties, upload images, 
and manage bookings through a responsive and user-friendly interface.

# 🚀 Features

- 🔐 User Authentication (Login & Signup)
- 🏠 Create, Edit & Delete Listings
- 📸 Image Upload with Cloudinary
- 💬 Flash Messages & Validation
- 📱 Responsive Design
- 🗺️ Interactive Maps
- 🛡️ Secure Authentication using Passport.js

# 🛠️ Tech Stack

- Node.js
- Express.js
- MongoDB Atlas
- Mongoose
- EJS
- Bootstrap
- Passport.js
- Cloudinary
- Render
